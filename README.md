# Hexapawn
## 1. 내용
알파 베타 프루닝 기법을 이용하여 최대 최소 알고리즘을 최적화시킨 헥사폰 게임 구현 <br><br>

## 2. 구성 요소
1) 3x3 보드판
2) 검은색 폰x3
3) 흰색 폰x3
<br>

## 3. 기본 규칙
1) 위치 : 반대편 진영에 횡대 배열 (default)
2) 이동 : 앞, 옆
3) 공격 : 왼쪽 대각선 앞, 오른쪽 대각선 앞
<br>
